# 🛒 E-Commerce Product Listing & Cart System

A full-stack **E-Commerce Desktop Application** that allows users to browse products, add items to cart, and manage purchases, while providing admin controls for product management.

---

## 🚀 Features

### 👤 User Authentication
- User Registration & Login
- Input validation (Email must contain `@gmail.com`)
- Secure session handling
- Logout functionality with proper redirection

### 🛍️ Product Catalog
- View all products with categories
- Product details page
- Search and filter functionality

### 🛒 Shopping Cart
- Add / Remove products
- Update product quantity
- View total price dynamically

### 🛠️ Admin Panel
- Admin login
- Add / Update / Delete products
- Manage product listings
